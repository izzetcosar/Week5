# Pratik - Array
**Bu pratikte aşağıdaki 4 aşamayı kodlamanız isteniyor.**

- 10 adet tam sayı verisi alacak bir dizi tanımlayınız.

- Bu dizinin elemanlarını bir for döngüsü ile doldurup, foreach döngüsü ile tekrar ekrana yazınız.

- Bu diziye kullanıcıdan alınan yeni bir değeri ekleyiniz (11. eleman olarak)

- Bu diziyi büyükten küçüğe ekrana yazdırınız.
